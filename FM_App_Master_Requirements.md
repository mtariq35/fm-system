## 4. PHASE 1 — QR CODE VISITOR FEEDBACK SYSTEM

### 4.1 Purpose
Replace the current manual/verbal complaint process with a digital, QR-code-based feedback system placed in washrooms and facility areas. Visitors scan, submit feedback, and the team is notified instantly.

---

### 4.2 Visitor-Facing Feedback Form

**Trigger:** Visitor scans QR code posted in washroom or facility area

**Languages:** English + Arabic (with language switcher)

**Rating Input:**
- Simple 3-button rating: ✅ Good | 😐 Neutral | ❌ Bad

**Issue Categories (multi-select):**
- 🧹 Cleaning / Hygiene
- 🔧 Damage or Repair Needed
- ⚠️ Safety Concern
- 🧴 Supplies Running Out (soap, paper, etc.)

**Additional Fields:**
- Description / Comments (free text)
- Photo Upload (optional — visitor can attach one or more photos)
- Contact Details (optional — visitor can choose to leave name + phone/email for follow-up)

**On Submission:**
- Confirmation screen shown to visitor ("Thank you, your feedback has been received.")
- If contact details provided — optional follow-up possible from team

---

### 4.3 Notification & Routing

| Rule | Detail |
|---|---|
| Who gets notified | All 3 supervisors simultaneously |
| Notification method | Dashboard alert (real-time) |
| Notification content | Issue type, rating, description, photo (if any), timestamp, location |

---

### 4.4 Complaint Lifecycle / Status Tracking

Each complaint follows this workflow:

```
OPEN → ASSIGNED → IN PROGRESS → VERIFIED → CLOSED
```

| Status | Meaning |
|---|---|
| Open | Complaint received, not yet assigned |
| Assigned | Supervisor has assigned it to a team member |
| In Progress | Team member is actively working on it |
| Verified | Supervisor has verified the resolution |
| Closed | Issue fully resolved and closed |

---

### 4.5 Supervisor / Team Dashboard

**Access:** Supervisors and Facility Manager login to web dashboard

**Dashboard Features:**
- Live feed of all incoming complaints
- Complaint cards showing: issue type, rating, photo, timestamp, current status
- Assign complaint to team member
- Update status through lifecycle stages
- Filter by: status, issue type, date, rating
- Search complaints

**Facility Manager View (additional):**
- All supervisor activity visible
- Override / reassign any complaint
- Access to all reports

---

### 4.6 Reporting (Facility Manager)

**Real-Time Dashboard:**
- Total complaints today / this week / this month
- Breakdown by issue type
- Average resolution time
- Open vs. closed complaint counts
- Rating trends (Good / Neutral / Bad over time)

**Periodic Reports:**
- Daily Summary Report (auto-generated each day)
- Weekly Performance Report (auto-generated each week)
- AI-generated narrative summary of trends and recommendations
- Exportable (PDF / print)

---

### 4.7 QR Code

- One unique QR code per location/area (e.g., Washroom A, Washroom B, etc.)
- QR code links directly to the feedback form for that specific location
- Location is auto-tagged on each submission
- Printable QR code poster generated by the system
